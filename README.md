# China SSL Journal

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为 HTML 文档的存储与展示空间。

## 项目简介

本项目旨在收集、整理并托管一系列独立的 HTML 页面。这些页面可能包含技术文档、学习笔记、工具演示或其他静态内容，以纯前端形式呈现，无需后端支持。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── example1.html
│   └── example2.html
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件视为一个独立页面，文件名应简洁明了。
- `assets/`：存放页面间共享的样式、脚本及图片资源。

## 页面归档说明

- 所有页面均为纯 HTML + CSS + JavaScript，不依赖外部数据库或服务器端处理。
- 页面内容由仓库维护者定期更新或新增，归档时保持原有结构与功能。
- 若页面引用外部资源（如 CDN 库），请确保其稳定可用，并在文档中注明来源。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来建议新增或更新页面内容。
- 提交新页面时，请将 HTML 文件放入 `pages/` 目录，并确保路径引用正确。
- 若页面需要额外资源（如图片、字体），请一并放入 `assets/` 对应子目录。
- 请勿在页面中包含任何形式的营销、诱导点击或违反平台政策的内容。
